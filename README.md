# 📊 Linear Regression Visualizer: Understanding How Machines Learn from Data

## 📌 Overview
This project demonstrates how machines learn patterns from data using **Linear Regression**, one of the simplest and most fundamental machine learning algorithms.

The goal is to show how a model can:
- Learn the relationship between variables  
- Fit a straight line to data  
- Make predictions  
- Measure its accuracy using error  

---

## 🧠 Concept

Linear Regression is based on the equation:

y = mx + b

Where:
- **m** = slope (rate of change)  
- **b** = intercept (starting point)  

The model finds the best values of **m** and **b** to fit the given data.

---

## ⚙️ Features

-  User input for custom datasets  
-  Visualization of data points  
-  Best fit regression line  
-  Mean Squared Error (MSE) calculation  
-  Prediction for new input values  
-  Highlighted predicted point on graph  

---

## 💻 How It Works

1. User enters X and Y values  
2. The model calculates:
   - Slope (m)
   - Intercept (b)  
3. A regression line is generated  
4. Error (MSE) is calculated  
5. User can input a new X value to get prediction
